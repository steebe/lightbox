# Light Table iOS App

A simple, elegant light table app for iOS designed for scanning and digitizing photography negatives.

## Features

- **Adjustable Brightness**: Control the backlight intensity from 30% to 100%
- **Alignment Grid**: Toggle a grid overlay to help line up negative images
- **Adjustable Grid Spacing**: Control grid density from 25pt to 100pt
- **Collapsible Controls**: Hide/show controls with a tap for maximum screen space
- **Clean Interface**: Minimal, distraction-free design

## Requirements

- iOS 16.0 or later
- Xcode 15.0 or later

## Setup

1. Open `light-table.xcodeproj` in Xcode
2. Select your development team in the project settings (Signing & Capabilities)
3. Connect your iOS device or use the simulator
4. Build and run (⌘R)

## Usage

### Basic Usage
1. Launch the app - you'll see a bright white screen
2. Use the brightness slider to adjust the backlight intensity
3. Perfect for placing negatives on your screen to scan/photograph

### Using the Alignment Grid
1. Toggle "Alignment Grid" on to show grid lines
2. Adjust the grid spacing slider to set how far apart the lines are
3. Use the grid to line up your negatives straight and evenly spaced
4. Toggle the grid off when you don't need it

### Hiding Controls
- Tap the chevron button at the bottom to hide/show controls
- This gives you maximum screen space for scanning

## Use Case

This app is perfect for:
- Scanning film negatives with a digital camera
- Digitizing old negative collections
- Quick negative reviews and contact sheets
- Any situation where you need a bright, even backlight with alignment guides
